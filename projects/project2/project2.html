Residential burglary in Boston. Spatiotemporal analysis and geovisualization using R (2015–2022)

Context

The topic of this project is based on the fact that crime is an issue that is becoming increasingly important today.

Objectives

The objective is to use geospatial data and R to analyze residential burglary patterns in Boston (2015–2022).

Tools

• Language R R libraries: sf, spatstat, tmap, ggplot2, dplyr, leaflet

• Environment: RStudio

• Open data (geospatial + attribute) - Urban boundaries (Harvard University)
- Road network (Boston Open Data)
- Crime data 2015–2022 (Analyze Boston)

Tasks

• Data processing Data import, cleaning, preprocessing, CSV → geospatial conversion

• Spatial analysis KDE, NKDE, hotspot analysis, LISA spatial autocorrelation

• Visualization Mapping, clustering, temporal analysis, animation (2015–2022)

Results

Applications

This analysis can be applied to urban planning, retail location optimization, transport systems, and risk management.
